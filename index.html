Kennel Keeper

Engineering

Kennel Keeper is a volunteer task manager for shelters. It allows admin to view which volunteer is on shift, what general or animal specific tasks they've completed, and have yet to complete.

Kennel Keeper was built with small shelters as it's focus. Shelters that survive based on volunteering lack the tools and capital to have a system to manage shelter volunteers, pets, and tasks. Kennel Keeper provides a simple shelter manager allowing the shelters days to be streamlined and productive as possible.

This was a collabrative project. My responsiblities included building the react components, calling from the database, and general front end functionality.
